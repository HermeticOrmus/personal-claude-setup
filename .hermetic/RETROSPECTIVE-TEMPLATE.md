# Hermetic Retrospective Template

## How to Use This Template

**Weekly:** 15 minutes, focus on Quick Reflection
**Monthly:** 30 minutes, full Hermetic Audit
**Quarterly:** 60 minutes, deep alignment review + course correction

**Location:** Copy this template to:
- `~/dev/retrospectives/YYYY-MM-DD-weekly.md`
- `~/dev/retrospectives/YYYY-MM-monthly.md`
- `~/dev/retrospectives/YYYY-QN-quarterly.md`
