## Entry Points by Use Case

### "I'm starting a new project"

```
START → /gnosis (understand deeply)
     → strategic-plan-architect (create plan)
     → /dev-docs (preserve context)
     → [Build with principles]
     → /verify (before shipping)
     → /celebrate (when complete)
```

---

### "I'm facing an ethical dilemma"

```
START → /align (check alignment)
     → CONFLICT-RESOLUTION.md (navigate tension)
     → ECONOMIC-REALITY.md (if financial pressure)
     → [Make conscious choice]
     → Document decision
     → Retrospective (learn from it)
```

---

### "My code is a mess"

```
START → filesystem-organizer agent
     → apply-correspondence skill (structure reflects values)
     → /verify (prove organization)
     → Maintain rhythm (weekly/monthly/quarterly)
```

---

### "I'm learning something new"

```
START → Learning/ directory structure
     → Learning Master agents (Feynman, Tesla, etc.)
     → apply-mentalism (build mental models)
     → Ship projects (Vibration principle)
     → Retrospective (extract learnings)
```

---

### "I'm burning out"

```
START → STOP WORKING (seriously)
     → Read Rhythm principle
     → Read ECONOMIC-REALITY.md
     → Retrospective (identify causes)
     → Course correct
     → Resume at sustainable pace
```

---

### "Client wants dark patterns"

```
START → /align (recognize extraction)
     → CONFLICT-RESOLUTION.md (response protocol)
     → Offer alternatives
     → IF accepted → Build empowering version
     → IF rejected → Document objection OR decline
```
