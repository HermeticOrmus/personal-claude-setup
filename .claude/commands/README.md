## Using Slash Commands

### Basic Syntax
```bash
/command-name [optional-context]
```

### Examples
```bash
# Invoke mentalism for new project
/mentalism user-authentication-system

# Balance competing priorities
/polarity speed-vs-quality

# Create momentum on stalled work
/vibration onboarding-flow

# Honor sustainable rhythms
/rhythm sprint-planning
```

### Command Flow
1. **Invoke** - Type the slash command
2. **Context** - Provide relevant context
3. **Ceremony** - Follow the guided steps
4. **Integration** - Apply insights to work
5. **Reflection** - Note what emerged
