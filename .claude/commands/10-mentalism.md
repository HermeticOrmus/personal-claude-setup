# Mentalism - Mind Clarity Ceremony
**"The All is Mind; The Universe is Mental"**

## Sacred Principle
Everything begins in consciousness. Clear thinking creates clear outcomes. Before building anything in the material world, perfect it in the mental realm.

## When to Invoke

**Use /mentalism when:**
- Starting new projects with unclear vision
- Code/architecture feels messy or confusing
- Team doesn't understand what you're building
- Experiencing mental fog or analysis paralysis
- Before major decisions requiring clarity
- When external chaos reflects internal confusion

## The Ceremony

### Phase I: Mental Emptying
```
1. Brain dump everything in your mind about this project/problem
2. No filtering, no judgment - get it all out
3. Write for 10 minutes straight
4. Empty the noise onto paper
```

### Phase II: Essence Crystallization
```
1. Answer in ONE clear sentence: "What is the ESSENCE of what I'm creating?"
2. Remove all complexity, find the core
3. If you can't explain simply, you don't understand deeply
4. Refine until crystal clear
```

### Phase III: Mental Model Mapping
```
1. Draw the key entities and their relationships
2. Visualize data flows and transformations
3. Map state changes and events
4. Create a mental picture so clear you could build from memory
```

### Phase IV: Coherence Validation
```
1. Ask: "Can I explain this to a 10-year-old?"
2. Check for contradictions in the model
3. Verify elegant simplicity
4. Ensure intuitive sense
5. If NO to any → Refine mental model
```

### Phase V: Material Alignment
```
1. Align implementation with mental model
2. File structure mirrors concepts
3. Function names match mental entities
4. Architecture reflects mental clarity
5. Code becomes thought made visible
```

## Expected Manifestation

**Immediate:**
- Crystal clear understanding of what you're building
- Simple, elegant mental model
- Confidence in approach
- Ability to communicate vision clearly
- Team alignment on shared understanding

**Long-term:**
- Clean architecture reflecting clear thought
- Reduced confusion and miscommunication
- Faster development (clarity breeds speed)
- Easier maintenance (clear models stay clear)
- Mastery through conscious creation

## Hermetic Truth

> "Confused thinking → Confused code. Clear mind → Clear creation."

Your external reality is a perfect mirror of your internal mental state. Master your mind, master your world.

Before any creation:
