## Skills in This Category

### n8n-wizard
**Purpose**: Automated n8n workflow import with complete database and documentation setup

**Use when**: Importing n8n workflows from JSON, setting up Supabase schemas, creating production automation systems

**Time**: 15-20 minutes per workflow (vs 45+ manual)

**Key Features**:
- Automated workflow import via n8n API
- Supabase database schema creation via MCP
- Comprehensive documentation generation
- Memory capture for future improvements
- Troubleshooting pattern recognition

**Activation**:

- File patterns: `**/n8n-workflows/**/*.json`, `*workflow*.json`

## Adding New Automation Skills

When creating a new automation skill:

1. **Create skill file**: `automation/skill-name.md`
2. **Follow format**: See `n8n-wizard.md` as template
3. **Register in**: `~/.claude/skill-rules.json`
4. **Include**:
   - When to use / when not to use
   - Prerequisites
   - Step-by-step process (with time estimates)
   - Success criteria
   - Common pitfalls
   - Hermetic alignment

5. **Update this README**
