# Ceremonial Agile - Sacred Sprint Practice

**Hermetic Principle**: Rhythm - "Transform process into conscious ritual"

**When to use**: Sprint planning, daily standups, sprint reviews/retros

---

## Quick Concept

**Standard Agile** = Mechanical process (checkbox exercise)

**Ceremonial Agile** = Conscious ritual (transformation container)

**Difference**: Intention, presence, wisdom extraction

**Result**: Sustainable pace + Deeper learning + Better code

---

## The 6+1 Sprint Structure

**Traditional Agile**: 2-week sprint (10 working days, no rest)

**Ceremonial Agile**: 6-day sprint + 1 integration day

### The Sacred Week

**Days 1-5 (Mon-Fri)**: Sprint work (building with intention)
**Day 6 (Sat)**: Harvest + Integration (extract wisdom)
**Day 7 (Sun)**: Sabbath (complete rest, unconscious integration)

**Why this works**:
- Sustainable (rest prevents burnout)
- Learning (integration extracts wisdom)
- Natural rhythm (mirrors creation story, circadian patterns)

## Common Workflows

### Workflow 1: Quick Start (First Sprint)

**Don't do everything at once** - Start small:

#### Week 1: Sprint Opening Only

**Monday 9 AM** (15 min - shortened version):
```markdown
## Sprint Opening - Sprint 1 (Simplified)

**Grounding**: 3 breaths together

**Intention**: "This sprint, we're building [goal]"

**Personal commitments** (each person):
- I commit to [one thing]

**Begin sprint**
```

**Friday check**: "How did that feel? Should we continue?"

---

#### Weeks 2-4: Add Daily Alignment

**If team liked Opening**:
- Replace standard standup with Daily Alignment
- Use Three-Part Check-In (Yesterday/Today/Feel)
- Keep 15-min timebox

---

#### Weeks 5-8: Add Harvest & Integration

**If Daily Alignment working**:
- End sprint with Harvest (Saturday AM)
- Add Integration (Saturday PM)
- Try 6-day sprint + Day 7 rest

**Full Ceremonial Agile now established**

---

### Workflow 2: Remote Team Adaptation

**Challenges**: Can't light candle together, harder to sense energy

**Solutions**:

#### Visual Anchors
- Share screen with candle image
- Timer visible to all
- Intention text on shared doc

#### Grounding
- "Everyone take 3 breaths, we start when all cameras are on"
- Use silence (don't fill with talk)

#### Talking Stick
- Use "raise hand" feature
- Facilitator calls on people in order
- Prevents crosstalk

#### Breakout Rooms
- For pair discussions during Integration
- Small group shadow work (then share with whole team)

#### Shared Documentation
- Live note-taking (Google Doc visible to all)
- Collaborative extraction
- Everyone can add insights

---

### Workflow 3: Large Team (10+ people)

**Problem**: Too many people for intimate ceremonies

**Solution**: Split into sub-teams

#### Sub-Team Ceremonies
- Teams of 4-6 people
- Each does full ceremonies
- Extract patterns to team wiki

#### All-Hands Sync
- Brief (30 min) after sub-team ceremonies
- Each team shares:
  - Sprint intention
  - Key patterns discovered
  - Support needed from other teams

**Benefit**: Intimate sub-teams + Cross-team alignment

## Facilitation Quick Guide

### Rotating Facilitator (Recommended)

**Why rotate**: Everyone learns facilitation, shares responsibility

**Rotation schedule**:
- Week 1: Person A facilitates all ceremonies
- Week 2: Person B facilitates all ceremonies
- Week 3: Person C facilitates all ceremonies
- Cycle continues

**Handoff**: Brief 10-min meeting Friday (outgoing teaches incoming)

---

### Facilitation Checklist

**Before ceremony**:
- [ ] Review ceremony structure (this skill file)
- [ ] Prepare timer (visible to all)
- [ ] Create shared doc for notes
- [ ] Set up candle/bell (or visual anchor if remote)

**During ceremony**:
- [ ] Arrive grounded (can't hold space if scattered)
- [ ] Model vulnerability (share truthfully first)
- [ ] Protect equal airtime (notice who speaks/doesn't)
- [ ] Don't rush discomfort (silence is okay)
- [ ] Manage time gently (visible timer, kind reminders)

**After ceremony**:
- [ ] Save notes to appropriate file
- [ ] Extract patterns to MOC/wiki
- [ ] Create Anki cards from key insights
- [ ] Update dev docs Layer 3 if applicable

## Troubleshooting

### Issue: "Ceremonies feel awkward"

**Normal**: First 2-3 sprints feel weird (breaking old patterns)

**Action**:
- Acknowledge discomfort (don't dismiss)
- Start small (just Opening Ceremony first)
- Let results speak (teams report better connection after 3 sprints)
- Give it 6 sprints before deciding

---

### Issue: "We don't have time"

**Calculate current waste**:
- Meetings that could be emails: 2 hrs/wk
- Fixing bugs from rushed work: 4 hrs/wk
- Repeated questions: 3 hrs/wk
- Context switching: 5 hrs/wk
- **Total waste: 14 hrs/wk**

**Ceremony investment**: 5 hrs/wk

**ROI**: Invest 5 hours, save 14+ hours = 9 hour net gain

---

### Issue: "Team resistant"

**Don't force** - Invite and demonstrate

**Approach**:
1. "I'd like to try something for 3 sprints. If it doesn't help, we stop."
2. Start with Opening Ceremony only (least threatening)
3. Ask for feedback after each sprint
4. Add more ceremonies only if team wants

**Reality**: Most teams want to continue after experiencing benefits

---

### Issue: "Facilitation is hard"

**Normal**: It's a learnable skill

**Practice**:
- Use this skill file as script (don't improvise at first)
- Rotate facilitators (everyone learns together)
- Debrief after ceremonies ("What worked? What didn't?")
- Get better over 10+ sprints (be patient with learning)
